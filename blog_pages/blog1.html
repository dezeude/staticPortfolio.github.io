HTML Canvas Experiments

HTML Canvas is the element that is used to draw 2D or 3D (using WebGL) graphics on websites. This element allows the client script in JavaScript to render shapes, images, lines, text, and more. To access the canvas' power we must use JavaScript code to render anything we want. The more I researched about the canvas element, the more I realized how powerful it really is. After seeing multiple implementations online, I am convinced that canvas is the key to creating unique websites that stand out. This blog post represents the fruits of my research of the element and what I can do with it.
